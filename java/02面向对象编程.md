# 面向对象编程

## 方法

1.设计

```java
//不带参数 不带返回值   大清国
	public void add1() {
		
	}
	
	//带参数  不带返回值
	//一个方法可以携带很多参数   貔貅
	public void add2(int a,double b) {
		
	}
	
	//带返回值 不带参数
	//一个方法只能有一个返回值   冤种
	public String add3() {
		
		//break  continue
		return "我爱您中国";
	}
	
	//带参数 也带返回值  商人
	public int add3(int a,int b) {
		
		return a+b;
	}
```



2.参数和返回值

一个方法能携带多个参数  参数就是写在参数体中的变量  用于接收方法外部的数据在方法内部使用

一个方法只能拥有一个返回值  返回值就是修改void为要返回数据的数据类型  用于把方法内部的数据返回到方法的外部



3.调用

```java
public static void main(String[] args) {
		
		One  one  = new One();
		
		//调用add1  
		one.add1();
		//调用add2  IBM  pc  一个人用电脑了   诞生了网络
		//必须给参数赋值 实参  形式  值  变量 表达式  另一个方法的返回值
		one.add2(100, 123.123);
		
		//调用add3()
		//调用了add3()直接得到了一个String
		//调用带返回值的方法需要接收返回值
		String a = one.add3();
		
		//调用add4()
		int num = one.add4(200,600);
		
		
	}
```

## 类的划分

类都是类  作用都一样

但是根据实际开发我们会把类划分成三种

1.测试类

​		作用就是运行代码 进行测试 程序的入口

```java
package model;

//个人习惯 XXXTest 测试类
public class Test {

	//作用运行
	public static void main(String[] args) {
		
	}
}

```



2.工具类

​		作用就是提供业务逻辑和公共的算法  （提供各种方法）

```java
package model;

//个人习惯 XXXUtils
public class Utils {
	
	//提供获取系统时间
	
	public void getSystemDate() {
		
	}
	
	//提供获取随机数
	public void getNum() {
		
	}
	
	//提供抽签
	public void getDataByDataSet() {
		
	}
	

}

```



3.实体类

​	利用的就是类(家庭)和对象(家庭中的人)

​	作用就是用来建模 得到现实生活中的一类实物

```java
package model;

//个人习惯 直接用实体的名字作为类名
public class Dog { //表示的事狗这类事物  所有的狗
	
	//面向对象
	//使用成员变量来描述这类事物的公共属性
	String name;
	int age;
	String color;
	
	//使用普通方法来描述这类事物的公共功能
	public void cry() {
		System.out.println("汪汪叫!");
	}
	
	public void eat() {
		System.out.println("在吃饭!");
	}
	
	public void sleep() {
		System.out.println("在睡觉!");
	}

}

```

实体类只要new对象  对象就是实体  一个具体的事物就出现了
