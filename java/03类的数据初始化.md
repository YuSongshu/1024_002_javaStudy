## 类的数据初始化（背）

类的数据初始化这个类执行new对象代码之后就能使用的数据 （类的初始化数据）

初始化代码  A   a = new A();

方案

```java
public class A {
	
	//类的构成   1.主方法  2.成员变量 3.局部变量  4.构造方法（参与对象的创建）  
	//5.普通方法  6.静态代码块（最先执行  只执行一次）
	
	//用到2和6
	
	public A() {
		System.out.println("初始化数据 构造方法");
	}

	static {
		System.out.println("初始化数据 静态代码块");
	}
}

```
