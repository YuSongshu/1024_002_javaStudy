## 运算符

java语言  数据类型多 运算符 

1.算术的  +  -  * / %  ++ +1  --  -1    数字类型 byte short int long  float double   组成的表达式 数字类型

2.关系的  >  <  >=  <=  ==  != 布尔类型 boolean

组成就是布尔类型的表达式  if(布尔类型的结果)   while(布尔类型的结果)

3.位运算  &  | ~   ^ 数字（与， 或， 非， 取异）

​	1.口诀

​		& 全1则1 否则为0

​		|  全0则0 否则位1

​		~  遇1则0  遇0则1

​		^ 相同为0  不同为1

​	2.位运算  二进制  按位运算  需要把参与运算的数字转换成 二进制

​		10->2  2->10

​	3.二进制数在运算  结果  二进制 -> 10进制

4.逻辑运算  &&   ||   ！ 布尔类型 boolean 组成就是布尔类型的表达式 

cj>=60 && cj<=70

```java
package one;

public class One {

	
	public static void main(String[] args) {
		/*3.位运算  &  | ~   ^ 数字
		1.有口诀的
		& 全1则1 否则为0
		|  全0则0 否则位1
		~  遇1则0  遇0则1
		^ 相同为0  不同为1*/
		
		int a = 9;
		int b = 3;
		System.out.println(a&b);
		
		//1.参与运算的10进制数 ->2进制
		//2.用2进制数参考口诀进行运算
		//3.把2进制的运算结果->10进制
		
		
		//10->2
		//9/2   当商==0的时候就结束 求 余数
		//9/2 = 4 = 1
		//4/2 = 2 = 0
		//2/2 = 1 = 0
		//1/2 = 0 = 1
		//00001001
		
		//3/2 = 1 = 1
		//1/2 = 0 = 1
		//00000011
		
		//& 全1则1 否则为0
		//a&b
		//00001001
		//00000011
		//00000001 -> 10进制  = 1
		
		//|  全0则0 否则位1
		//00001001
		//00000011
		//00001011 = 11
		
		//~  遇1则0  遇0则1
		//~a
		//00001001
		//11110110
		
		//^ 相同为0  不同为1
		//00001001
		//00000011
		//00001010 = 10
		
		//扩展
		//异或   c=a^b   b=a^c   a=b^c
		
		
//加密应用
		char r1 = '十';
		char r2 = '点';
		char r3 = '进';
		char r4 = '攻';
		
		char key = '8'; //密钥
		
		char m1 = (char)(r1 ^  key);
		char m2 = (char)(r2 ^  key);
		char m3 = (char)(r3 ^  key);
		char m4 = (char)(r4 ^  key);
		
		System.out.println(""+r1+r2+r3+r4);
		System.out.println(""+m1+m2+m3+m4);
		
	}
}

```

2.创建完成一个类可以什么都不写   什么都不写不代表什么没有

3.类中有一些代码但是我们看不见 （隐藏代码）

(背)

1.主方法

```java
//1.主方法  作用 运行代码	  是不是必须点写  不是
```

2.成员变量

```java
public class Two {
	//2.成员变量  作用  公共资源(*)  是不是必须点写  不是
```


## 概念 (背)

成员变量   指声明或定义在类中的变量 == 全局变量

局部变量   指声明或定义在方法中的变量包含方法的参数

普通方法   指一个独立的可调用的代码块可以封装很多零散的代码

构造方法  1.构造方法名字等于类名

​				  2.构造方法写不写都默认存在 （隐藏代码）

​				  3.构造方法用来创建一个类的对象

​				  4.构造方法不能拥有任何的返回值类型 

## 关键字(背)

void 表示或修改方法的返回值类型的一种 表示当前方法不返回任何数据

new 用来给引用数据类型赋值表示开辟一块独立的内存空间

static 用来修饰成员变量和普通方法表示静态的或类的
